#água #ordem

Cria um clone de água do usuário, com os mesmos atributos mas com vida diferente. O clone também copia a arma usada pelo usuário no momento em que a magia foi usada. Os clones dividem a iniciativa com o usuário.

Para cada 2 pontos de água, cria um clone.
Para cada ponto de algum elemento de ordem usado, aumenta em 7 de vida do clone
